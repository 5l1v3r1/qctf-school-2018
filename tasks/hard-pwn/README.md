# Service configuration

# HOW TO CONFIGURE?
 - specify path for tokens file in [configure.py](./configure.py)
 - run `python3 configure.py`

It will generate `main`, `getflag`, `check.py` and `generate.py` binaries in the respective directories.

# FILES FOR TASK
 - `libc.so.6`
 - `main`
 - `getflag`
