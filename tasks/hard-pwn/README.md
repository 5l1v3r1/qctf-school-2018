# Service configuration

# HOW TO CONFIGURE?
 - specify path for tokens file in (configure.py)[./configure.py]
 - run `python3 configure.py`

It will generate `main` and `getflag` binaries in the respective directories.

# FILES FOR TASK
 - `libc.so.6`
 - `main`
 - `getflag`
